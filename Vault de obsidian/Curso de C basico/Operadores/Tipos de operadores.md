- Los siguentes son los operadores mas conocidos

```c
	//Operadores
	=   //Asignacion
	==  //Comparacion
	!=  //Distinto 
	>   //Mayor
	<   //Menor
	>=  //Mayor igual
	<=  //Menor igual
	+   //Suma
	*   //Multiplicacion
	-   //Resta
	/   //Division
	%   //Modulo
	&&  //and
	||  //or
	!   //not
	/=  //Dividir y asignar   C /= a  es igual C = C / a
	%=  //Modulo y asignar    C %= a es igual C = C % a 
	<<= //shift izquierdo    C<<=2 es igual a C = c << 2
	>>= //shift derecho
    &=  //bitwise and
    ^=  //bitwise exclusive or
    |=  //bitwise inclusive or
    ~   //Cambio de bits
```
